# KIRAENGINEER Trading Platform

Professional trading community platform for KIRAENGINEER, built with Next.js 15, React, Tailwind CSS, NextAuth, Prisma/PostgreSQL, Stripe, Resend, and TradingView-ready embeds.

## Quick Start

```bash
npm install
cp .env.example .env
npm run prisma:generate
npm run dev
```

## Main URLs

- `/` Home
- `/about` About Kira Engineer
- `/market-analysis` CMS-ready market analysis
- `/live-performance` performance dashboard
- `/vip-membership` Stripe-ready pricing
- `/education` education center
- `/economic-calendar` calendar page
- `/contact` Telegram/contact hub
- `/login` and `/register`
- `/dashboard` user dashboard
- `/admin` admin dashboard shell
- `/api/performance` performance API
